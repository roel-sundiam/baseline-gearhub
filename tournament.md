# Tennis Tournament Management Module

## Overview

Build a Tournament Management Module for a Tennis Court Reservation System. The module should support both admin and member users with a clean, mobile-friendly interface.

---

## 1. Admin Tournament Configuration

### Create Tournament

- Input tournament name
- Select tournament type:
  - Singles
  - Doubles

### Manage Matches

- Create and organize match pairings
- Assign players or teams
- Structure matches into rounds (e.g., elimination format)

### Schedule Matches

- Set match dates and time slots
- Display in calendar or schedule format

### Update Scores

- Input match results and scores
- Automatically progress winners to next round
- Track match status (upcoming, ongoing, completed)

### Declare Winners

- Finalize tournament results
- Identify champion, runner-up, and placements

---

## 2. Member Access

Members can:

- View tournaments
- View schedules
- View match pairings
- Check scores and results
- See winners

---

## 3. Player Ranking System

### Singles Points

- Champion: 100 points
- Runner-up: 70 points
- Semi-finalist: 40 points
- Quarter-finalist: 20 points
- Participation: 10 points

### Doubles Points

- Champion: 80 points per player
- Runner-up: 50 points
- Semi-finalist: 30 points
- Quarter-finalist: 15 points
- Participation: 5 points

---

## 4. Ranking Logic

- Players accumulate points from tournaments
- Rankings are based on total points
- Leaderboard sorted highest to lowest
- Rankings update after each tournament

---

## 5. General Requirements

- Clean and modern UI
- Mobile-friendly design
- Easy navigation
- Real-time or near real-time updates (if possible)

---

## Goal

Deliver a complete tournament module for managing tournaments, viewing schedules, tracking results, and maintaining rankings.
